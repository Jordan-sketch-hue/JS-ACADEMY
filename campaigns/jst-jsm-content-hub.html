Voice & production notes
Voice brief
Smooth, soft, high. Natural Jamaican inflection where it fits — not performed. Think: a confident woman explaining something clearly to a friend, not a voiceover artist. Record in one clean take per reel. No reverb. Light compression if needed.
On-screen text
Bold white text, centered, appearing in sync with VO. Key phrases only — not full sentences. Use the hook lines from each reel as the primary text overlays. Add subtle fade-in, no bounce animations.
Sound layer
No AI sounds. Use a royalty-free ambient track under the VO — something clean and minimal. TikTok sound library: search "calm background" or "soft ambient." Keep music at 15–20% volume so VO leads clearly.

TikTok Content Posting API
Can we automate TikTok posting?
Straight answer: yes, with setup. Here's exactly what's possible and what the path looks like.
Current status
TikTok Content Posting API is real — but gated.
TikTok has an official Content Posting API that lets apps upload videos directly to your account. The catch: it requires OAuth login through TikTok's developer portal and app approval. I cannot connect to it from this environment directly, but you can set it up — here's the exact path.
API exists & is documented
Requires developer app setup
Can't auth from this session
1
Create a TikTok Developer App
Go to developers.tiktok.com → My Apps → Create App. Set app type to "Web." You need a business account linked. This gives you a client_key and client_secret.
2
Enable Content Posting API scope
Inside your app settings, add the scope video.upload and video.publish. TikTok reviews this — approval takes 1–3 business days for personal creator accounts, longer for business apps targeting others.
3
OAuth login flow (one time)
Your app redirects you to TikTok login, you authorize it, and it returns an access_token. This token lets scripts post on your behalf. Valid for 24 hours — refresh tokens last 365 days.
4
Upload video via API
POST to open.tiktokapis.com/v2/post/publish/video/init/ with your video file URL, caption, and privacy settings. TikTok handles the encoding. Your reel goes live (or to draft for review first if you prefer).
5
Build the automation here (JST Academy)
Once you have the access token, I can write a Node.js or Python script inside this repo that takes a video file + caption and posts it automatically. We can schedule posts via cron, trigger from a content calendar, or build a simple dashboard for it — your call.
⚠
The "open in browser" limitation: TikTok's API requires an OAuth authorization flow — it can't be done by reading your open browser tab directly. That's a TikTok security requirement, not a technical limitation on my end. The one-time login through their portal is the step I need you to complete. Once you have a token, everything else can be automated from here.
Easier alternative while you set up the API
Buffer or Later — both connect to TikTok Business accounts and let you schedule video posts from a dashboard. Free tiers available. You upload the reel + caption here, set a post time, done. Connecting these to the JST Academy automation system is straightforward once the API token is in place.
